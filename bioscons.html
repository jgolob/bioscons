bioscons.infernal.cmalign_mpi_action(target, source, env)[source]¶

Run cmalign using mpi (mpirun)

Number of processors (default 2) is set using ‘cmalign_nproc’. Note that ‘cmalign –mpi’ is only available if cmalign is configured using the ‘–enable-mpi’ on compilation.

target - [file in stockholm format, file containing align scores] source - [alignment profile, fasta file]
